# 🔐 Design a Modern Login Form Using Tailwind CSS

A simple and responsive login form built using **HTML5** and **Tailwind CSS**. This project demonstrates how to create a clean and modern user interface with utility classes provided by Tailwind CSS.

---

## 👨‍💻 Author

**Sagar Kirtakar**

---

## 📖 About

This project showcases a modern login form with a clean layout and responsive design. It includes input fields, a remember me checkbox, a forgot password link, and a login button with hover effects.

---

## ✨ Features

- ✅ Responsive design
- ✅ Email and password input fields
- ✅ Remember me checkbox
- ✅ Forgot password link
- ✅ Login button with hover effects
- ✅ Register account link
- ✅ Rounded corners and shadow effects
- ✅ Focus styles for better user experience
