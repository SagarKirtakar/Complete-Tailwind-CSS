# 🎨 Tailwind CSS Simplified

A beginner-friendly repository to learn **Tailwind CSS** with simple explanations and examples.

## 📌 About

This repository contains my notes and practice examples while learning Tailwind CSS. It covers the most commonly used utility classes and helps in building responsive and modern user interfaces quickly.

## 🚀 Topics Covered

* Typography
* Colors
* Background Colors
* Margin & Padding
* Width & Height
* Borders & Border Radius
* Flexbox
* Grid
* Buttons
* Hover Effects
* Shadows
* Positioning
* Display Classes
* Responsive Design

## 📂 Project Structure

```text
Tailwind-CSS-Simplified/
│
├── index.html     # Main HTML file with Tailwind examples
├── styles.css     # Custom CSS styles (if any)
├── notes.md       # Tailwind CSS notes and explanations
└── README.md      # Project documentation
```

## 💡 Example

```html
<button class="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-700">
    Click Me
</button>
```
